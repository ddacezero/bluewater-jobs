# Jobs at Bluewater Resorts

An Applicant Tracking System (ATS) built with React + Vite for managing the hiring pipeline at Bluewater Resorts.

## Features

- **Dashboard** — Overview of hiring stats, pipeline chart, and recent activity
- **Candidates** — Full candidate table with search, filters, recruiter assignment
- **Pipeline** — Kanban board view across 8 hiring stages
- **Jobs** — Create, edit, and manage job postings
- **Reports** — Per-job and per-month hiring analytics with charts
- **Talent Pool** — Pooled candidates from closed positions, ready to reactivate

### Sample Job Openings (Hospitality)
Chef de Partie (Hot), Demi Baker, Baker, Plumbing Supervisor, Carpenter, Front Office Supervisor, Housekeeping Team Leader, Commis Chef, Spa Therapist

### Hiring Stages
Applied → Screening → Initial Interview → Exam → Departmental Interview → Final Interview → Job Offer → Hired

### Candidate Features
- Add/edit skills
- Add notes
- Upload/download resumes
- Endorse for other positions
- Recruiter assignment (Joela / Ranie)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is configured for one-click deployment on [Vercel](https://vercel.com).

## Tech Stack

- React 18
- Vite 6
- Inline CSS (no external UI library)
- DM Sans (Google Fonts)
